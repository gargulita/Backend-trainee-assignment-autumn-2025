##  Запуск проекта

### 1. Запуск PostgreSQL и сервиса
Проект использует Docker Compose.  
Для старта базы данных и backend-сервера 

```bash
docker-compose up --build
```
После запуска сервис доступен по адресу:
http://localhost:8080
